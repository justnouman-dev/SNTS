CSV Format Guide

Your CSV file must have a header row with exactly these column names (order does not matter):

Column Name Required Description Example
name Yes Food name (unique) Dal Makhani
calories_per_100g Yes kcal per 100g 155
protein_per_100g Optional Protein grams per 100g 8.5
carbs_per_100g Optional Carbohydrate grams per 100g 22.0
fats_per_100g Optional Fat grams per 100g 3.5

Sample CSV content: 

name,calories_per_100g,protein_per_100g,carbs_per_100g,fats_per_100g

Import Rules

  • Duplicate food names (same name already in database) are skipped automatically.
  • Name matching is case-insensitive — "Dal" and "dal" are treated as the same food.
  • Rows with missing or non-numeric calorie values are skipped with a warning.
  • Optional macro columns default to 0 if left blank.
  • File must be saved as UTF-8 encoding (standard for Google Sheets / Excel CSV export).
  • There is no row limit, but very large files may take a few seconds.
